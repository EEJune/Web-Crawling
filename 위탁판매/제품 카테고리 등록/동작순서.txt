동작순서
1.대카테고리명 기준("/")있는지 판단
1-1 있다면 "/"을 기준으로 대카테고리를 나눈다 ex)세제/구강>세제 구강 후에 다시 1
1-2 대카테고리명 일치가 없다면->2
1-3 일치하는것이 존재시 행의 값을 일치하는 범위로 나눠서 2

2.중카테고리명 기준("/")있는지 판단
2-1 있다면 "/"을 기준으로 중카테고리를 나눈다 ex)뷰티소품/미용재료>뷰티소품 미용재료 후에 다시2
2-2 중카테고리명 일치가 없다면->3
2-3 일치하는것이 존재시 행의 값을 일치하는 범위로 나눠서 3

3.소카테고리명 기준("/")있는지 판단
3-1 있다면 "/"을 기준으로 소카테고리를 나눈다 ex)비누받침/자석비누홀더 후에 다시 3
3-2 소카테고리명 일치가 없다면
3-2-2 4단계 카테고리로 간다
3-2-3 4단계 카테고리에서 1개만 일치시 쿠팡 카테고리 return
3-2-4 4단계 카테고리에서 여러개 일치시 글자수 까지 일치하는 것으로 return
3-3 일치하는 것이 지정된 행의 범위내에 1개만 존재시 쿠팡 카테고리를 가져온다 return
3-3-2복수로 존재시 4단계 카테고리로 
3-3-3 4단계 카테고리에서 1개만 일치시 쿠팡 카테고리 return
3-3-4 4단계 카테고리에서 여러개 일치시 글자수 까지 일치하는 것으로 return

끝까지 없을 시 쿠팡카테고리=-1 일것이다 그럴경우 None표시
